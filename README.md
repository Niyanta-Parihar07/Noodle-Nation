# 🍜 Noodle Nation

Noodle Nation is a simple and interactive recipe website that showcases a variety of delicious noodle dishes from around the world.

🌐 Features

- 🍱 6 Different Noodle Recipes
- 📖 Detailed Ingredients & Step-by-Step Instructions
- 🧱 Card-Based Layout
- 📱 Responsive Design (Mobile Friendly)
- 🎨 Clean and Modern UI using CSS
- 🏷️ Recipe Categories (Veg / Non-Veg)
